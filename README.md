# pipr-detector
Resumen de la Arquitectura y el Stack Tecnológico: Detalla cómo interactúan Flask, SQLite, OpenCV, YOLOv8 y Pandas.  Explicación Técnica del Algoritmo: Explica cómo tu código de visión artificial procesa las imágenes (preprocesamiento CLAHE, círculos de Hough, análisis de contornos y puntuación de consenso mediante Intersección sobre Unión/IoU).  Mapa del Árbol de Directorios: Detalla la estructura de carpetas, incluyendo dónde residen detection.png y los archivos multimedia subidos.  Tabla de Documentación de Endpoints: Mapea visualmente cada ruta URL, método de petición HTTP, descripción y lógica subyacente.  Instalación y Configuración Paso a Paso: Guía a los usuarios para crear un entorno virtual, instalar las dependencias (incluyendo los requisitos de lectura/escritura de Excel como openpyxl) y crear el esquema de la base de datos.  Instrucciones de Ejecución: Documenta cómo exponer la aplicación en tu red de área local (apuntando a tu referencia comentada de la red local xxxxxxxx), junto con las instrucciones para capturar los conteos de inventario en vivo a través del bucle interactivo de la ventana de la cámara de OpenCV (usando el activador de la tecla 'q').
